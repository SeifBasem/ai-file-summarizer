# 🧠 AI Text Summarizer (BERT-based)

This project is an AI-powered text summarization system using a pretrained model from Hugging Face.

It reads files (.txt, .pdf, .docx) and generates a concise summary.

---

## 🚀 Features

- Supports multiple file types:
  - TXT
  - PDF
  - DOCX
- Handles long documents using chunking
- Uses pretrained model (T5-small)
- Saves summary automatically

---

## 🛠️ Installation

```bash
pip install -r requirements.txt
```
